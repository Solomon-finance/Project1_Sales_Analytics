# Project 1 - Sales Performance Analytics Dashboard

## Overview
Analysis of 9,800 retail sales records to uncover 
business insights using Python and Pandas.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Key Findings
- West region leads in sales at $710,219
- Technology is the top category at 36.6%
- Consumer segment drives 50.8% of revenue
- November shows highest monthly sales
- Phones are the top selling sub-category

## Dataset
Superstore Sales Dataset from Kaggle
